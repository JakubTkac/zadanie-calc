# zadanie-calc
Kalkulacka je rozdelena na 2 casti /frontend /backend

na frontend bol pouzity JS kniznica React na backend nodeJS framework NestJS

# Instalacia

## 1.

1. cd backend
2. npm i
3. npm run start

4. localhost:3000

## 2.

1. cd frontend
2. npm i
3. npm run build
4. npm run preview

5. localhost:4173
